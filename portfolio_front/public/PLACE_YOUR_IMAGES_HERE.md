# Required image assets

Place these files in this `public/` folder:

1. **me.jpg**     — your profile photo (square, ~600x600px recommended)
2. **logo.png**   — your logo for the navbar (transparent PNG, ~38px tall, wide aspect)
3. **favicon.png** — browser tab icon (32x32 or 64x64 px)

If `me.jpg` is missing, a placeholder avatar icon is shown automatically.
If `logo.png` is missing, the navbar falls back to showing "Mesfin Maru." as text.
